AngularJS学习笔记 - Day 1

2016-07-23

阅读并学习angular官方文档中angular-function

function

1. angular.bind

Returns a function which calls function fn bound to self (self becomes the this for fn). You can supply optional args that are prebound to the function. This feature is also known as partial application, as distinguished from function currying.

  • 作用：返回一个新的函数，绑定这个函数的this指向self
  • 使用：angular.bind(self, fn, args);
  • 参数：
    • self：新函数的上下文对象
    • fn：需要绑定的函数
    • args：传递给函数的参数
  • 返回值：this指向self的新函数

2. angular.bootstrap

Use this function to manually start up angular application.

  • 作用：手动启动Angular应用
  • 使用：angular.bootstrap(element, modules, config);
  • 参数：
    • self：DOM元素，Angular应用的根
    • modules：一个需要注入程序的模型的数组。数组中的每一项都应该是一个预定义的模块的名称，或者是一个将被injector调用的配置函数。
    • config： 一个用于定义应用配置项的对象，它支持以下键：
      • strictDi - disable automatic function annotation for the application. This is meant to assist in finding bugs which break minified code. Defaults to false.
  • 返回值：新创建的应用的依赖注入。

3. angular.copy

Creates a deep copy of source, which should be an object or an array.

  • 作用：深度拷贝对象或者数组。
  • 使用：angular.copy(source, [destination]);
  • 参数：
    • source：将用于拷贝的源，可以为任意类型，包括null undefined。
    • destination：被复制的对象，如果提供了，则类型须与源数据类型一致。
  • 返回值：拷贝或更新后的destination。

4. angular.element

Wraps a raw DOM element or HTML string as a jQuery element.

  • 作用：将DOM元素或者HTML字符串封装为jquery对象。
  • 使用：angular.element(element);
  • 参数：
    • element：HTML字符串或者是DOM元素。
  • 返回值：jquery对象。

    If jQuery is available, angular.element is an alias for the jQuery function. If jQuery is not available, angular.element delegates to Angular’s built-in subset of jQuery, called “jQuery lite” or jqLite.

如果引入了jquery，则返回一个jquery对象；如果没有引入jquery，则使用angular内置的简化版的jquery - jqLite

Angular’s jqLite：它只提供了下列jquery方法：

  • addClass() - Does not support a function as first argument
  • after()
  • append()
  • attr() - Does not support functions as parameters
  • bind() - Does not support namespaces, selectors or eventData
  • children() - Does not support selectors
  • clone()
  • contents()
  • css() - Only retrieves inline-styles, does not call getComputedStyle()`. As a setter, does not convert numbers to strings or append ‘px’, and also does not have automatic property prefixing.
  • data()
  • detach()
  • empty()
  • eq()
  • find() - Limited to lookups by tag name
  • hasClass()
  • html()
  • next() - Does not support selectors
  • on() - Does not support namespaces, selectors or eventData
  • off() - Does not support namespaces, selectors or event object as parameter
  • one() - Does not support namespaces or selectors
  • parent() - Does not support selectors
  • prepend()
  • prop()
  • ready()
  • remove()
  • removeAttr()
  • removeClass() - Does not support a function as first argument
  • removeData()
  • replaceWith()
  • text()
  • toggleClass() - Does not support a function as first argument
  • triggerHandler() - Passes a dummy event object to handlers
  • unbind() - Does not support namespaces or event object as parameter
  • val()
  • wrap()

4. angular.equals

Determines if two objects or two values are equivalent. Supports value types, regular expressions, arrays and objects.

确定两个对象/值是否完全相等，支持，值，正则表达式，数组，对象。

Two objects or values are considered equivalent if at least one of the following is true:

  • Both objects or values pass === comparison.
  • Both objects or values are of the same type and all of their properties are equal by comparing them with angular.equals.
  • Both values are NaN. (In JavaScript, NaN == NaN => false. But we consider two NaN as equal)
  • Both values represent the same regular expression (In JavaScript, /abc/ == /abc/ => false. But we consider two regular expressions as equal when their textual representation matches).

During a property comparison, properties of function type and properties with names that begin with $ are ignored.

至少符合下列一个条件，两个对象或者值才相等。

  • 两个对象和值都通过===判断。
  • 两个对象和值类型相同，且属性都通过angular.equals比较相等。
  • 两个值都为NaN（在javascript中，NaN == NaN => false，但在angular中是相等的）。
  • 两个值都为同一个正则表达式（在javascript中，/ABC/ == /ABC/ => false，当两个正则表达式匹配相同文本时，angular考虑他们是相等）。

在属性比较中，以$开头的属性和函数将会被忽略。

  • 作用：比较两个对象/值。
  • 使用：angular.equals(o1, o2);
  • 参数：
    • o1,o2：需要进行比较的两个参数。
  • 返回值：true/false。

5. angular.extend

Extends the destination object dst by copying own enumerable properties from the src object(s) to dst. You can specify multiple src objects. If you want to preserve original objects, you can do so by passing an empty object as the target: var object = angular.extend({}, object1, object2).

从src中拷贝已有的属性，再拓展至dst中。

  • 作用：相当于合并多个对象，src中相同的属性会被覆盖。需要深度拷贝的，使用angular.merge代替。
  • 使用：angular.extend(dest, src);
  • 参数：
    • dest：需要合并到的目的对象。
    • src：源对象。
  • 返回值：包含src中所有属性的dest对象。

6. angular.foreach

Invokes the iterator function once for each item in obj collection, which can be either an object or an array. The iterator function is invoked with iterator(value, key, obj), where value is the value of an object property or an array element, key is the object property key or array element index and obj is the obj itself. Specifying a context for the function is optional.

调用迭代器去遍历obj中每一项，obj可以是一个对象或者数组。可以传入一个迭代时调用的函数，传入参数有3个：value, key, obj。value是迭代时的每一项的值，key是键，在函数中指迭代对象本身。

  • 作用：遍历一个对象，并执行相应操作。
  • 使用：angular.forEach(obj, iterator, [context]);
  • 参数：
    • obj：遍历的对象。
    • iterator：遍历函数。
    • context：若传入，则在遍历函数中，this指向该对象。
  • 返回值：obj。

7. fromJson

Deserializes a JSON string.

反序列化json。

  • 作用：将json解析为object。
  • 使用：angular.fromJson(json);
  • 参数：
    • json：json串。
  • 返回值：object。

8. angular.identity

A function that returns its first argument. This function is useful when writing code in the functional style.

返回第一个参数的函数。在函数代码风格（函数式编程？）中有用。

  • 作用：返回第一个参数。
  • 使用：angular.identity(value);
  • 参数：
    • value：json串。
  • 返回值：value本身。

9. angular.injector

Creates an injector object that can be used for retrieving services as well as for dependency injection (see dependency injection).

  • 作用：创建一个依赖注入。
  • 使用：angular.injector(modules, [strictDi]);
  • 参数：
    • modules：A list of module functions or their aliases. See angular.module. The ng module must be explicitly added.
    • strictDi：Whether the injector should be in strict mode, which disallows argument name annotation inference.(default: false)
  • 返回值：注入器对象。

10. angular.isArray/isDate/isDefined/isElement/isFunction/isNumber/isObject/isString/isUndefined

  • 作用：判断是否为该类型。
  • 使用：angular.isArray(value);
  • 参数：
    • value：需要判断的值。
  • 返回值：如果是该类型则返回true。

其中：

  • isElement判断该元素是否为DOM元素或jquery对象
  • isNumber在遇到NaN, +Infinity 和 -Infinity 时都返回true
  • isObject中，null返回false

11. angular.merge

Unlike extend(), merge() recursively descends into object properties of source objects, performing a deep copy.

和前面的extend()不一样，merge()是会对源的对象属性进行递归深拷贝。

  • 作用：相当于合并多个对象，src中相同的属性会被覆盖。需要深度拷贝的，使用angular.merge代替。
  • 使用：angular.merge(dest, src);
  • 参数：
    • dest：需要合并到的目的对象。
    • src：源对象。
  • 返回值：包含src中所有属性的dest对象。
